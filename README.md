# IdeaVault Client

IdeaVault Client is a simple startup idea sharing platform where users can browse ideas, view details, add their own ideas, and interact through comments.

## Main Theme

The main theme of this project is collecting, sharing, and managing startup ideas in one place.

## How It Works

- Users open the frontend on `https://ideavault-client-ten.vercel.app/`.
- The frontend connects to the backend API on `https://ideavault-server-gold.vercel.app/`.
- Users can explore ideas, view trending ideas, add new ideas, update their own ideas, delete ideas, and comment on ideas.

## Tech Stack

- Next.js
- React
- Better Auth
- MongoDB
